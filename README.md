##### 文件类型

- 零件
- 装配体

##### 软件布局

-  菜单栏

- 工具栏：绘制草图时的常用工具

  - 草图
  - 特征
  - 评估

- 设计树

  - FeatureManager
    - 零件特征管理器
      - 基准面
        - 前、上、右 基准面
        - 可以调节或恢复基准面大小
      - 原点
        - 左下角的原点坐标系显示方向
        - 画面中的原点坐标系只显示 X 箭头（短）与 Y 箭头（长）
        - 插入-参考几何体-坐标系

- 零件视窗

  - 零件
  - 基准面

- 任务列表

  

##### 画正方体

- 草图
  - 草图绘制
  - 选择基准面
  - 智能尺寸
  - 草图由蓝色变为黑色，说明完全约束
- 拉伸凸台
  - 选择方向
  - 确认尺寸
