### 草图编辑功能

- 裁剪实体
  - 强劲裁剪
    - 按住鼠标，鼠标划过的实体都会被裁剪，裁剪范围为最近的端点处
    - 按住shift，可以延长线段
  - 边角
  - 在内剪除
  - 在外剪除
  - 裁剪到末端
    - 点击线段裁剪到最近端点
- 转换实体引用
  - 将几何中已有的线提取到目前正在绘制的草图中
  - 可以理解为转换草图引用，转换轮廓引用
  - 在绘制草图时，通过转换实体引用，可以获取到别的面上的图案
  - 所选的面必须要平行与当前的草图面
- 等距实体
  - 通过等距实体可以创建一个相同的元素，并设置距离

## 特征

### 特征成型时的技巧

#### 所选轮廓

- 草图有多个封闭轮廓时，特征成型时
  - 所选轮廓选项默认空白时是全选，也可以单独选择轮廓生效

#### 厚度为零

- 比如拉伸切除时，如果切除图案与成型图案相切，会导致相切部分厚度为零
  - 实体必须有厚度
- 方法
  - 画出相切部分的厚度草图
  - 先不相切，画好草图后切除，再设置草图相切，不会报错，但这样的加工没有意义

#### 特征父子关系

- 注意特征的父子关系
  - 某些特征的修改，会影响到基于该特征的子特征，比如位置关系



### 构建实体

- 拉伸凸台/基体

  - 通过草图拉伸构建实体
    - 桌子

  - 旋转凸台/基体
    - 通过草图旋转构建实体
      - 杯子
    - 需要封闭的草图，
      - 如果没有封闭，系统会自动规划封闭的线，或者自动勾选薄壁特征
    
  - 扫描
    - 通过将草图沿着轨迹扫描构建实体
      - 回形针

- 放样凸台
  - 连接两个不同的草图，形成实体，比如梯形体块
