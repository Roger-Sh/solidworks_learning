## 草图

### 草图元素

- 直线
  - 几何关系
  - 构造线
  - 无限长度
- 矩形
  - 边角矩形
  - 中心矩形
- 圆
  - 中心圆
    - 确定圆心，半径
  - 周边圆
    - 三点确定一个圆
  - 约束
    - 通过圆心确认位置约束
- 槽
  - 直槽口
  - 中心点直槽口
  - 三点圆弧直槽口
- 圆弧
  - 圆心弧
  - 切线弧
- 圆角
  - 注意参数，圆角半径不能超过边的长度
- 多边形
- 样条曲线
- 椭圆



### 草图几何关系

- 重合
  - 线与线重合
  - 端点与线重合
- 中点
- 相切
- 平行
- 相等
- 共线
- 对称
  - 需要一条构造线作为对称轴，选中构造线后，选择构造线两边的图形



### 草图编辑功能

- 裁剪实体
  - 强劲裁剪
    - 按住鼠标，鼠标划过的实体都会被裁剪，裁剪范围为最近的端点处
    - 按住shift，可以延长线段
  - 边角
  - 在内剪除
  - 在外剪除
  - 裁剪到末端
    - 点击线段裁剪到最近端点
- 转换实体引用
  - 将几何中已有的线提取到目前正在绘制的草图中
  - 可以理解为转换草图引用，转换轮廓引用
  - 在绘制草图时，通过转换实体引用，可以获取到别的面上的图案
  - 所选的面必须要平行与当前的草图面
- 等距实体
  - 通过等距实体可以创建一个相同的元素，并设置距离



### 智能尺寸

- 竖直尺寸
- 水平尺寸
- 长度尺寸
  - 与线垂直方向
- 角度
  - 选中两条线确定夹角
- 从动与驱动尺寸
  - 从动表示受其他约束被动改变的尺寸
  - 驱动尺寸表示主动由用户定义的尺寸



### 草图绘制思路

- 一般草图需要完全约束
- 先绘制直线，注意平行，垂直，水平，竖直等关系
- 标注尺寸
- 如何确定什么尺寸没有标注
  - 通过拖动蓝色线段，观察其运动方式，确定需要标注的尺寸



### 草图镜像与阵列

- 镜像实体

  - 要镜像的实体
  - 镜像轴

- 草图阵列

  - 线性草图阵列
    - 可以设置阵列的方向，数量，距离
  - 圆周草图阵列
    - 可以设置阵列圆心，角度，数量，间距等

- 移动实体

  - 移动
  - 复制
  - 缩放
  - 旋转
  - 伸展

  

## 特征

- 构建实体

  - 拉伸凸台/基体

    - 通过草图拉伸构建实体
      - 桌子

    - 旋转凸台/基体
      - 通过草图旋转构建实体
        - 杯子

    - 扫描
      - 通过将草图沿着轨迹扫描构建实体
        - 回形针

  - 放样凸台
    - 连接两个不同的草图，形成实体，比如梯形体块

- 切除实体

  - 拉伸切除

    - 通过草图拉伸切除实体

    - 旋转切除
      - 通过草图旋转切除实体

    - 扫描切除
      - 通过将草图沿着轨迹扫描切除实体

  - 放样切割
    - 连接两个不同的草图，切除实体

- 异形孔向导

  - 螺纹孔
  - 螺钉间隙孔
  - 销钉孔
  - 沉头孔

- 曲线
  - 螺旋线，用来绘制弹簧
- 螺纹
  - 对圆柱体 插入-注解-装饰螺纹线 选择圆柱体的底边
